ShelterFind

The ‘ShelterFind’ is a mobile application which serves as a mediator between; hostels and students, ‘homestel’ owners and ‘homestel’ searchers, hotels and accommodation seekers, house owners and house searchers and finally, room owners and room searchers, by creating a connection between these pair of dynamic duos that actually need each other to operate well and complete. Owners can provide information about their properties and facilities using a web portal through the app. This information would act as the bases for the search results displayed to the user. Owners would have individual accounts where they can manage information that should be displayed to users. Users would be provided with a simple-to-navigate interface to view facilities of available owners where facilities can be viewed or filtered by various categories like; price, location, type, etc. Users would be given the chance to rate owners based on their previous experience with the owner’s services which would serve as a guide for new users. An administrator would also use the web portal in order to administer the system and keep information accurate. An administrator can for instance verify property owners and manage user information. In order to make this system highly secured, a google image search would be done on each image an owner uploads to make sure that no image is downloaded from somewhere and brought in to the system. This is meant to stop fraudsters from duping users.
